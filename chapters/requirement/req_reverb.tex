\reqPrefix{Reverb}
\section{Reverb}
Based on \autoref{sec:reverberation}, the following requirements are made for the \gls{reverb} effect:

\begin{requirement}\label{req:reverb1}
    \requirement{The \gls{reverb} effect must at least produce 1000 echoes per second.}
    \argument{Functional requirement for room simulation\citep{natural_sounding_revorb}.}
\end{requirement}

\begin{requirement}\label{req:reverb2}
    \requirement{The user must be able to change the ratio between the direct input and the output of the \gls{reverb} effect.}
    \argument{Design choice to make the effect flexible for the user (wet/dry).}
\end{requirement}

\begin{requirement}\label{req:reverb3}
    \requirement{The user must be able to increase the number of echoes per second.}
    \argument{To make the user able to choose the room characteristics of the effect.}
\end{requirement}
